# Assets Setup

## Current Status
The app is configured to run **without** custom assets. This allows you to test the app immediately.

## Adding Custom Assets (Optional)

If you want to add custom icons and splash screens later:

1. **App Icon** (`assets/icon.png`):
   - Size: 1024x1024 pixels
   - Format: PNG
   - No transparency

2. **Splash Screen** (`assets/splash.png`):
   - Size: 2048x2048 pixels (recommended)
   - Format: PNG
   - Will be centered on white background

3. **Android Adaptive Icon** (`assets/adaptive-icon.png`):
   - Size: 1024x1024 pixels
   - Format: PNG
   - Safe area: Keep important content in center 512x512

4. **Web Favicon** (`assets/favicon.png`):
   - Size: 48x48 or 96x96 pixels
   - Format: PNG or ICO

## To Enable Assets

Once you have the images, update `app.json`:

```json
{
  "expo": {
    "icon": "./assets/icon.png",
    "splash": {
      "image": "./assets/splash.png",
      "resizeMode": "contain",
      "backgroundColor": "#ffffff"
    },
    "android": {
      "adaptiveIcon": {
        "foregroundImage": "./assets/adaptive-icon.png",
        "backgroundColor": "#ffffff"
      }
    },
    "web": {
      "favicon": "./assets/favicon.png"
    }
  }
}
```

## Quick Asset Generation

You can use online tools to generate these assets:
- [AppIcon.co](https://www.appicon.co/)
- [Expo Asset Generator](https://www.npmjs.com/package/expo-asset-generator)

Or use Expo's built-in asset generation:
```bash
npx expo-asset-generator
```
