# üì± How to Start Android Emulator

## Quick Steps

### Step 1: Open Android Studio

1. **Launch Android Studio** from Start Menu

### Step 2: Open Device Manager

1. **Click "More Actions"** (top right)
2. **Select "Virtual Device Manager"**
   - Or: Go to **Tools ‚Üí Device Manager**

### Step 3: Start Your Emulator

1. **Find your device** in the list (e.g., "Pixel 5 API 33")
2. **Click the Play button** (‚ñ∂Ô∏è) next to it
3. **Wait for emulator to boot** (2-3 minutes first time)

### Step 4: Test Your App

1. **Once emulator is running** (you'll see Android home screen)
2. **Go back to your terminal**
3. **Press `a`** to open app on Android
4. **App should load!** üéâ

## üêõ If You Don't See Any Devices

**You need to create one first:**

1. **In Device Manager, click "Create Device"**
2. **Choose "Pixel 5"**
3. **Download system image** (API 33 or 34)
4. **Click "Finish"**
5. **Then start it** (Play button)

## üí° Pro Tip

**Keep the emulator running** - you don't need to close it. Just press `a` in terminal whenever you want to test your app!

## ‚úÖ That's It!

Once emulator is running, pressing `a` in terminal will automatically open your app on it!
